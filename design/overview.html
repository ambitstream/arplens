ArpLens

Recreate standard arpeggios from audio.

Get ready-to-use arpeggiator settings for your favorite synth, plugin or DAW.

One page, one workflow

  1. Upload — drag & drop or browse (MP3 / WAV / M4A)
  2. Waveform — Focus Region, then a 3–20 s loop selection
  3. Analyze — one primary button with progress state
  4. Results — BPM, Input Notes, Style, Rate, Octaves, Sequence, Confidence
  5. Edit — notes, style, rate, octaves, BPM ×2/÷2
  6. Preview — looping Tone.js playback (Play / Pause / Loop)

Components

  • Header · UploadCard · Waveform (FocusRegion, LoopSelection, PlaybackControls)
  • AnalyzeButton · ResultPanel (BPMCard, NotesEditor, StyleEditor, RateEditor, OctavesEditor, SequenceView, ConfidenceBadge)
  • PreviewPlayer · PlaybackController

Non-negotiables (frozen spec v2.1)

Honest UI: unknown values show Not detected — never placeholders.

Confidence is High / Medium / Low with green / yellow / red — the text label is always visible; color is never the only channel.

Sharp note names only (C# D# F#). Subtle animations. Keyboard accessible.

Desktop-first, mobile-friendly; everything stacks, no functionality removed.
